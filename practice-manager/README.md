# Milestone Practice Page Manager

This tool is only used to manage practice pages maintained in Milestone
project.

## Installation

```sh
git clone https://github.com/briansalehi/practice-manager.git
cd practice-manager
make link
```

## Usage

```sh
practice-manager -h
```

## Removal

```sh
git clone https://github.com/briansalehi/practice-manager.git
cd practice-manager
make unlink
```
